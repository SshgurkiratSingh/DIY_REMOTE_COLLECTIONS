# Receiver Firmware

Receiver projects grouped by control/input stack.

## Families
- `esp-now-legacy/`: early ESP-NOW receiver references.
- `direct-bts/`: direct motor output receivers.
- `direct-bts-mpu6050/`: direct motor output + IMU-aware receiver.
- `fsi6-bts/`: FS-i6 RC receiver based BTS bridge variants.
