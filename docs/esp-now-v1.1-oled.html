ESP_NOW_V1.1_OLED

Mature transmitter version with portal pairing, persistent settings, and rotary-driven OLED menu flows.

User interface model

The firmware exposes a full rotary encoder navigation stack over OLED pages for telemetry, targets, settings, and system information.

Portal target registration

Select Add Tgt(AP) to start AP mode and register a receiver MAC from a browser at 192.168.4.1.

Persistent settings

Critical parameters are saved via NVS so reboot does not reset field tuning.

deadzone, centerX, centerY, invertX, invertY, txRateHz, rxEnabled

Telemetry callback path

Optional receive callback allows reply packet parsing to confirm link quality and carry compact sensor feedback.

Build / Flash

cd firmware/transmitter/esp-now-oled/v1.1
pio run
pio run --target upload
